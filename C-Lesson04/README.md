# Урок по C №04. Управляющие конструкции. Циклы

## Синтаксис оператора ```while```

```c
statement x;
while (condition)
{
	statement block;
}
statement y;
```

## Синтаксис оператора ```do-while```

```c
statement x;
do
{
	statement block;
} while (condition);
statement y;
```

## Синтаксис оператора ```for```
```c
for (initialization; condition; increment/decrement/update)
{
	statement block;
}
statement y;
```
